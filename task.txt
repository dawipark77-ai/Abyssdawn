한글로 답해

지금 작업 목표:
기존 스킬트리 연결 로직/데이터는 
절대 건드리지 않고 UI 레이어만 새로 교체할거야.

---

1단계 — 먼저 파악만 해줘. 수정 절대 하지 말고.

프로젝트에서 아래 항목들 전부 찾아서 정리해줘.

1. 스킬트리 관련 스크립트 전체 목록
   - 각 스크립트가 어떤 역할을 하는지
   - 어떤 스크립트가 어떤 스크립트를 참조하는지

2. SkillSO (또는 스킬 데이터 ScriptableObject) 구조
   - 어떤 필드가 있는지
   - 티어/선행조건이 어떻게 정의되어 있는지

3. 현재 스킬트리 UI 구조
   - 어떤 씬에 있는지
   - 어떤 GameObject/Prefab으로 구성되어 있는지
   - 무기 선택 버튼과 스킬트리 연결이 
     어떻게 되어있는지

4. 기존 SkillTreeManager 
   (또는 스킬트리 관리 스크립트)
   - public 메서드 목록
   - 어떤 이벤트/콜백을 사용하는지

파악 완료 후 정리해서 보여줘.
2단계 수정은 내가 확인하고 나서 따로 지시할게.
